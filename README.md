#  Gripper Bot (Manual Pick & Place Robot)

## Overview  
This project is a manually controlled gripper robot built completely from scratch.  
It is designed to pick and place objects weighing up to 500 grams using a custom-built gripping mechanism and wireless control.

The bot combines mechanical design, fabrication, and real-time control, making it a strong foundation project in robotics.

---

##  Features  
- Fully built from scratch  
- Custom-designed gripper mechanism  
- Manual wireless control using FlySky FS-i6  
- Can lift objects up to 500g  
- Strong and durable metal alloy chassis  
- Smooth pick-and-place operation  

---

##  Design & Development  

### Gripper Design  
- Designed using Fusion 360  
- Focused on grip strength and stability  

###  Chassis  
- Built using metal alloy sheet  
- Provides durability and structural strength  

### Control System  
- Controlled using FlySky FS-i6 transmitter and receiver  
- Enables real-time manual operation  

---

## Components Used  
- DC Motors / Servo Motors  
- Motor Driver  
- FlySky FS-i6 Transmitter & Receiver  
- Battery (Li-Po recommended)  
- Metal Alloy Sheet  
- Custom Gripper (Fusion 360 design)  

---

##  Working Principle  
1. The user controls the robot using the FS-i6 transmitter  
2. Signals are sent to the receiver  
3. Motors drive the movement and gripper mechanism  
4. The gripper opens and closes to pick and place objects
